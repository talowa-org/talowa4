## Getting Started

### Registration Process

1. **Receive a Referral Code**
   - Get a referral code from an existing Talowa member
   - The code will look like "TAL2B3C4D" (9 characters)
   - You can also scan a QR code shared by a member
   - **Auto-Fill Option**: Click a referral link to automatically fill the code

2. **Sign Up**
   - Visit the Talowa registration page or click a referral link
   - Enter your personal information
   - **Referral code auto-fills** if you clicked a referral link
   - Or manually input the referral code you received
   - Create your account credentials

3. **Complete Payment**
   - Choose your membership plan ($99.99 USD)
   - Select your preferred payment method
   - Complete the payment process
   - Your account will be activated automatically

4. **Get Your Referral Code**
   - Once activated, you'll receive your unique referral code
   - This code is yours to share with others
   - You can find it in your dashboard at any time

### Auto-Fill Registration System

Talowa makes registration easier with automatic referral code filling:

#### How It Works

1. **Click a Referral Link**
   - Someone shares a link like: `https://talowa.web.app/join?ref=TAL234567`
   - Click or tap the link on your device

2. **App Opens Automatically**
   - The Talowa app opens (or web page loads)
   - The referral code is extracted from the link

3. **Code Auto-Fills**
   - The registration form automatically fills in the referral code
   - You'll see a green notification: "Referral code auto-filled: TAL234567"

4. **Complete Registration**
   - Fill in your personal information
   - Complete the payment process
   - Your referral relationship is automatically established

#### Supported Link Formats

- **Web Links**: `https://talowa.web.app/join?ref=TAL234567`
- **App Links**: `talowa://join?ref=TAL234567`
- **QR Codes**: Scan QR codes containing referral links
- **Short Links**: Shortened versions of referral links

#### Benefits

- **No Manual Entry**: No need to remember or type referral codes
- **Error Prevention**: Eliminates typos in referral codes
- **Seamless Experience**: One-click registration process
- **Guaranteed Connection**: Ensures proper referral relationship

## Understanding the Role System

Talowa uses a 5-tier role system that determines your earning potential and team leadership responsibilities.

### Role Hierarchy

#### 1. Member (Starting Role)
- **Requirements**: 0+ referrals, 0+ team size
- **Benefits**: 
  - Basic commission structure
  - Access to referral tools
  - Team building resources

#### 2. Activist
- **Requirements**: 10+ direct referrals, 25+ total team size
- **Benefits**:
  - Increased commission rates
  - Advanced analytics
  - Leadership training access

#### 3. Organizer
- **Requirements**: 25+ direct referrals, 100+ total team size
- **Benefits**:
  - Higher commission multipliers
  - Team management tools
  - Regional recognition

#### 4. Coordinator
- **Requirements**: 100+ direct referrals, 500+ total team size
- **Benefits**:
  - Premium commission structure
  - Advanced team analytics
  - Leadership mentorship opportunities

#### 5. Regional Coordinator
- **Requirements**: 500+ direct referrals, 2500+ total team size
- **Benefits**:
  - Maximum commission rates
  - Regional leadership role
  - Exclusive events and recognition

### Role Progression

- **Automatic Advancement**: Roles are updated automatically when you meet requirements
- **Real-time Tracking**: Monitor your progress in the dashboard
- **Notifications**: Receive alerts when you're close to advancing
- **Celebration**: Role advancements are celebrated with achievements and recognition

## Troubleshooting

### Common Issues and Solutions

#### 1. Referral Code Not Working
**Problem**: Someone can't use your referral code
**Solutions**:
- Verify the code is correct (9 characters, starts with "TAL")
- Check if the code is still active
- Ensure the person hasn't already registered
- Contact support if issues persist

#### 2. Payment Processing Issues
**Problem**: Payment not going through
**Solutions**:
- Verify payment information is correct
- Check with your bank for any blocks
- Try a different payment method
- Contact payment support

#### 3. Commission Discrepancies
**Problem**: Commissions don't match expectations
**Solutions**:
- Review commission calculation rules
- Check if referrals completed payment
- Verify role-based multipliers
- Contact support with specific details

#### 4. Team Statistics Not Updating
**Problem**: Team numbers seem incorrect
**Solutions**:
- Allow 24 hours for updates
- Refresh your dashboard
- Check if team members are active
- Report persistent issues to support
